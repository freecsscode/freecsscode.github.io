Sejarah Singkat CSS!

CSS (Cascading Style Sheet) adalah standar pembuatan dan pemakaian style untuk dokumen terstruktur. CSS digunakan untuk mempersingkat pemakaian tag HTML seperti halnya font, color, text, dan table menjadi lebih ringkas sehingga tidak terjadi pengulangan tulisan. Penggunaan yang paling umum digunakan adalah untuk menformat halaman web yang ditulis dalam tag tag HTML maupun XHTML.

Meskipun demikian, bahasanya sendiri dapat digunakan untuk semua jenis dokumen XML termasuk juga SVG dan XUL. Spesifikasi CSS diatur oleh perusahaan World Wide Web Consortium (W3C). CSS pertama kali muncul dibawah lisensi W3C dimana bermula sejak munculnya SGML sekitar tahun 1970-an. Sejak kemunculannya tersebut, CSS mengalami perkembangan yang sangat pesat pada masa itu.

Mulai dari Format dasar CSS yang banyak kita gunakan sekarang ini merupakan ide dari seseorang programming bernama Hakom Wium Lie yang teruang dalam proposalnya mengenai CSS pada bulan Oktober 1994 dalam konferensi W3C di Chicagom Illinois. Kemudian Hakom bersama para rekannya yang bernama Bert Bos mengembangkan suatu standard CSS. Pada akhir dekade tahun 1996, CSS telah resmi dipublickasikan dan menyusul kemudian CSS level 1 pada bulan Desember dalam tahun yang sama. Apakah kalian ingin mencoba menggunakan CSS ?
