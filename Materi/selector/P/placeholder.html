::placeholder

(CSS : Selector)

selektor ::placeholder memilih elemen formulir dengan teks placeholder.

Teks placeholder diatur dengan atribut placeholder, yang menentukan petunjuk yang menjelaskan nilai yang diharapkan dari bidang input.

Tips : Warna default teks pengganti adalah abu-abu terang di sebagian besar browser.

# Syntax

CSS
input::placeholder {

color : blue ;

font-size : 13px ;

font-weight : 700 ;

}
