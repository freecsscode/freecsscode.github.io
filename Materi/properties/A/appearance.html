appearance

(CSS : Properties)

Properti appearance digunakan untuk menampilkan elemen menggunakan gaya platform-asli berdasarkan tema sistem operasi pengguna.

# Syntax

CSS
.thing {

-webkit-appearance : value ;

-moz-appearance : value ;

appearance : value ;

}

Ini digunakan karena satu dari dua alasan :

  1. Untuk menerapkan gaya khusus platform ke elemen yang tidak memilikinya secara default.
  2. Untuk menghapus gaya khusus platform ke suatu elemen yang memang memilikinya secara default.

Misalnya, input dengan pencarian type = "search" di browser WebKit secara default memiliki sudut yang membulat dan sangat ketat dalam hal apa yang dapat anda ubah melalui CSS. Jika anda tidak ingin gaya itu, anda dapat menghapusnya dalam satu gerakan dengan penampilan.

CSS
input [type=search] {

-webkit-appearance : none ;

}

Atau Anda dapat mengambil input dengan tipe = teks dan membuatnya seperti input pencarian :

CSS
input [type=text] {

-webkit-appearance : searchfield ;

}
