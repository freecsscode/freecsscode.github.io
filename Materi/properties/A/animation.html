Animation

(CSS : Properties)

Properti animasi di CSS dapat digunakan untuk menghidupkan banyak properti CSS lainnya seperti warna, warna latar, tinggi, atau lebar . Setiap animasi perlu ditentukan dengan @keyframe at-rule yang kemudian disebut dengan properti animasi, cotohnya seperti berikut:

# Syntax

CSS
.element {

animation : pulse 5s infinite ;

}

@keyframes pulse {
0% {
background-color : #72E0D1 ;
}
100% {
background-color : #FF4136 ;
}

Setiap @keyframes at-rule menentukan apa yang harus terjadi pada saat-saat tertentu selama animasi. Misalnya, 0% adalah awal dari animasi dan 100% adalah akhir. Keyframe ini kemudian dapat dikontrol oleh properti animasi singkatan, atau delapan sub-properti, untuk memberikan kontrol lebih besar tentang bagaimana keyframe tersebut harus dimanipulasi.

# sub properties

  • animation-name: mendeklarasikan nama @keyframe at-rule untuk memanipulasi.
  • animation-duration: lamanya waktu yang dibutuhkan untuk sebuah animasi untuk menyelesaikan satu siklus.
  • animation-timing-function: menetapkan kurva akselerasi preset seperti kemudahan atau linier.
  • animation-delay: waktu antara elemen yang dimuat dan awal urutan animasi.
  • animation-direction: menetapkan arah animasi setelah siklus. Standarnya diatur ulang pada setiap siklus
  • animation-iteration-count: berapa kali animasi harus dilakukan.
  • animation-fill-mode: menyetel nilai mana yang diterapkan sebelum / sesudah animasi.
    Misalnya, Anda dapat mengatur status animasi terakhir untuk tetap di layar, atau Anda dapat mengaturnya untuk beralih kembali ke sebelum ketika animasi dimulai.
  • animation-play-state: jeda / mainkan animasi.
