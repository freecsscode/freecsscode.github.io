align-self

(CSS : Properties)

Itu memungkinkan untuk mengesampingkan nilai align-item untuk item fleksibel tertentu.

Properti align-self menerima 5 nilai yang sama dengan align-item:

  • flex-start : batas mulai sumbu awal tepi dari item ditempatkan pada garis awal
  • flex-end : batas mulai sumbu awal tepi dari item ditempatkan pada garis akhir.
  • center : item dipusatkan pada sumbu silang.
  • baseline : item disejajarkan seperti garis belakang mereka disejajarkan.
  • stretch (default) : regangkan untuk mengisi wadah (masih mengikuti lebar minimum / lebar maks).

# Syntax

CSS

align-self : flex-start | flex-end | center | baseline | stretch

.flex-item {

align-self : flex-start ;

align-self : flex-end ;

align-self : center ;

align-self : baseline ;

align-self : stretch ;

}

# Demo

Demo berikut menunjukkan bagaimana suatu item dapat menyelaraskan dirinya dalam wadah fleksibel tergantung pada nilai align-selfnya :

  • Item pertama diatur menjadi flex-start
  • Item kedua diatur menjadi flex-end
  • Item ketiga diatur menjadi center
  • Item keempat diatur menjadi baseline
  • Item kelima diatur menjadi stretch
