flex-flow

(CSS : Properties)

Ini adalah singkatan untuk arah-fleksibel dan flex-wrap.

# Syntax

CSS
flex-flow: (flex-direction) || (flex-wrap)

.flex-container {

flex-flow : row wrap ;

}

Anda dapat menentukan satu atau dua nilai, apa pun urutannya

See the Pen flex-flow by Fahmi Ariq (@fahmi_ariq) on CodePen.

Kedua daftar berperilaku dengan cara yang sama persis :

  • Yang kuning memiliki flex-direction: row dan flex-wrap: wrap
  • Yang biru memiliki flex-flow: row wrap
