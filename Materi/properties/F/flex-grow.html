flex-grow

(CSS : Properties)

Properti flex-grow menentukan berapa banyak item akan tumbuh/bertambah relatif terhadap sisa item fleksibel di dalam wadah yang sama.

Catatan: Jika elemen tersebut bukan item yang fleksibel, properti flex-grow tidak berpengaruh.

# Syntax

CSS
div {

flex-grow : 1 (Nilai) ;

}
