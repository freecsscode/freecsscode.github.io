font-kerning

(CSS : Properties)

properti font-kerning mengontrol penggunaan informasi kerning yang disimpan dalam font.

Kerning adalah istilah dalam sebuah proses typography. Kerning tidak hanya mengacu pada jarak antara dua huruf saja, tapi juga didefinisikan sebagai proses penyesuaian space secara manual.

# Syntax

CSS
.contoh {

font-kerning : auto ;

font-kerning : normal ;

font-kerning : none ;

}
