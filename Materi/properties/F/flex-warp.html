flex-wrap

(CSS : Properties)

Properti flex-shrink menentukan bagaimana item akan menyusut relatif terhadap sisa item fleksibel di dalam wadah yang sama.

Catatan: Jika elemen tersebut bukan item yang fleksibel, properti flex-shrink tidak memiliki efek.

Properti flex-wrap menerima 3 nilai berbeda yaitu:

  • nowrap (default) : single-line yang dapat menyebabkan wadah berlebihan (overflow)
  • wrap : multi-baris, arah didefinisikan oleh arah fleksibel
  • wrap-reverse : multi-garis, berlawanan dengan arah yang ditentukan oleh arah fleksibel

# Syntax

CSS
flex-warp: nowarp | wrap | wrap-reverse

flex-contain {

flex-warp : warp ;

}
