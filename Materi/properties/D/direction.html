direction

(CSS : Properties)

Properti direction di CSS menetapkan arah aliran konten dalam elemen level blok. Ini berlaku untuk elemen teks, inline, dan inline-block. Itu juga mengatur perataan teks dan arah yang sel-sel tabel mengalir dalam baris tabel.

# Syntax

CSS
p.rtl {

direction : rtl ;

}

Nilai yang diterima adalah :

  • ltr - dari kiri ke Kanan, dan ini nilai adalah default
  • rtl - dari kanan ke kiri
  • inherit - mewarisi nilainya dari elemen induk.

Teks pada halaman ini diatur dalam arah ltr default. Kasus penggunaan yang paling umum untuk mengatur rtl adalah untuk halaman web dengan teks Ibrani atau Arab. Berikut ini adalah contoh dari bahasa Arab yang diatur dalam rtl:

طهيس يس تآخت تهات يس وريتتآن فروم ريغت تو لآفت تهات يس وسآد
