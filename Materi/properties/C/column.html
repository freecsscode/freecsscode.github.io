columns

(CSS : Properties)

Dengan hanya beberapa aturan CSS, Anda dapat membuat tata letak yang diikuti cetak yang memiliki fleksibilitas web. Ini seperti mengambil sebuah koran, tetapi karena kertasnya semakin kecil, kolom akan menyesuaikan dan menyeimbangkan secara otomatis yang memungkinkan konten mengalir secara otomatis.

# Syntax

CSS
.kolom {

-webkit-columns : 100px 3 ;

-o-columns : 100px 3 ;

columns : 100px 3 ;

}

Properti kolom akan menerima jumlah kolom, lebar kolom, atau kedua properti.

CSS

columns : column-width || column-count ;

Disarankan menggunakan penghitungan kolom dan lebar kolom untuk membuat tata letak multi-kolom yang fleksibel. Penghitungan kolom akan bertindak sebagai jumlah kolom maksimum, sedangkan lebar kolom akan menentukan lebar minimum untuk setiap kolom. Dengan menggabungkan sifat-sifat ini bersama-sama, tata letak multi-kolom akan secara otomatis terurai menjadi satu kolom pada lebar browser yang sempit tanpa perlu kueri media atau aturan lainnya
