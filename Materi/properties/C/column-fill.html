column-fill

(CSS : Properties)

Saat Anda menambahkan tinggi ke elemen multi-kolom, Anda dapat mengontrol bagaimana konten mengisi kolom. Konten dapat diseimbangkan atau diisi secara berurutan

CSS
.contoh1 {

-moz-column-fill : auto ;

column-fill : auto ;

}
.contoh2 {

-moz-column-fill : balance ;

column-fill : balance ;

}

Properti ini hanya tersedia di Firefox. Firefox akan secara otomatis menyeimbangkan konten dalam tata letak multi-kolom yang dibatasi ketinggian. Browser lain akan selalu mengisi kolom secara berurutan saat diberi batasan ketinggian.

Untuk membuat Firefox berperilaku seperti browser lain, yaitu untuk mengisi kolom secara berurutan, Anda dapat mengatur isi kolom dengan nilai "auto".
