background-color

(CSS : Properties)

Properti background-color dalam CSS menerapkan warna solid sebagai latar belakang pada suatu elemen. Ini adalah sebuah contohnya :

CSS
body {

background-color : #FF8A00 ;

}

Contoh yang digunakan di atas (#FF8A00) disebut kode hex, dan itu adalah salah satu dari beberapa cara yang CSS harus mewakili satu warna. Ada sejumlah cara untuk menemukan kode hex yang tepat. Siapa pun yang menggunakan alat desain telah melihat pemilih warna yang mirip dengan yang seperti ini :
