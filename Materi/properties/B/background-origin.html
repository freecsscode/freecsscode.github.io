background-origin

(CSS : Properties)

Properti background-origin menentukan di mana harus melukis latar belakang di seluruh elemen, baik di dalam perbatasan (border), atau di dalam padding.

# Syntax

CSS
.contoh {

background-origin : border-box ;

background-origin : padding-box ;

background-origin : content-box ;

}

Ada tiga nilai: border-box, padding-box, dan content-box. berikut adalah demonya :
