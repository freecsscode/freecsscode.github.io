background-position

(CSS : Properties)

Properti posisi latar belakang di CSS memungkinkan Anda untuk memindahkan gambar latar belakang (atau gradien) di dalam wadahnya.

# Syntax

CSS
html {

background-position : 100px 5px ;

}

Ini memiliki tiga jenis nilai:

  • Nilai Panjangnya (e.g. 100px 5px)
  • persentase (e.g. 100% 5%)
  • Kata Kuncinya (e.g. top right)

Nilai defaultnya adalah 0 0. Ini menempatkan gambar latar belakang Anda di kiri atas wadah.

Nilai panjangnya cukup sederhana: nilai pertama adalah posisi horizontal, nilai kedua adalah posisi vertikal. Jadi 100px 5px akan memindahkan gambar 100px ke kanan dan lima piksel ke bawah. Anda dapat mengatur nilai panjang dalam px, em, atau nilai panjang CSS lainnya.

Persentase bekerja sedikit berbeda. Dapatkan topi matematika Anda: memindahkan gambar latar belakang sebesar X% berarti itu akan menyelaraskan titik X% dalam gambar ke titik X% dalam wadah. Misalnya, 50% berarti akan menyelaraskan bagian tengah gambar dengan bagian tengah wadah. 100% berarti akan menyelaraskan piksel terakhir gambar dengan piksel terakhir wadah, dan seterusnya.

Kata kunci hanyalah pintasan untuk persentase. Lebih mudah untuk mengingat dan menulis atas kanan daripada 100% 0, dan itulah sebabnya kata kunci adalah sesuatu. Berikut adalah daftar kelima kata kunci dan nilai-nilai yang setara :

  • Atas: 0% vertikal semua
  • kanan: 100% secara horizontal
  • bawah: 100% secara vertikal
  • kiri: 0% secara horizontal
  • pusat: 50% horizontal jika horizontal belum ditentukan. Jika ya maka ini diterapkan secara vertikal.

Sangat menarik untuk dicatat bahwa tidak masalah urutan apa yang Anda gunakan untuk kata kunci: tengah atas sama dengan tengah atas. Anda hanya dapat melakukan ini jika Anda menggunakan kata kunci secara eksklusif. pusat 10% tidak sama dengan pusat 10%.
