background-clip

(CSS : Properties)

background-clip memungkinkan anda mengontrol seberapa jauh gambar atau warna latar belakang melampaui lapisan atau konten elemen.

CSS
.frame {

background-clip : padding-box ;

}

# Value

  • border-box adalah nilai default. Ini memungkinkan latar belakang untuk meluas sampai ke tepi luar dari batas elemen.
  • padding-box klip latar belakang di tepi luar padding elemen dan tidak membiarkannya meluas ke perbatasan.
  • content-box klip latar belakang di tepi kotak konten.
  • inherit menerapkan pengaturan klip latar belakang dari induk ke elemen yang dipilih.
