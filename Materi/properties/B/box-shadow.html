box-shadow

(CSS : Properties)

Properti box-shadow melampirkan satu atau lebih bayangan ke suatu elemen.

# Syntax

CSS
.contoh {

box-shadow : 5px 10px ;

box-shadow : 5px 10px red ;

box-shadow : 5px 5px 5px rgba(68, 68, 68, 0.6); ;

}

box-shadow : [batas horizontal] [batas vertikal] [blur radius] [warna] ;

  1. Offset horizontal (diperlukan) bayangan, positif berarti bayangan akan berada di sebelah kanan kotak, offset negatif akan menempatkan bayangan di sebelah kiri kotak
  2. Offset vertikal (diperlukan) dari bayangan, yang negatif berarti bayangan kotak akan berada di atas kotak, yang positif berarti bayangan akan berada di bawah kotak.
  3. Radius blur (diperlukan), jika diatur ke 0 bayangan akan tajam, semakin tinggi angkanya, semakin kabur bayangan itu, dan semakin jauh bayangan akan meluas. Misalnya bayangan dengan 5px horizontal offset yang juga memiliki radius blur 5px akan menjadi 10px total bayangan.
  4. Warna (wajib) - mengambil nilai warna apa pun, seperti hex, named, rgba, atau hsla. Jika nilai warna dihilangkan, bayangan kotak diambil dalam warna latar depan (warna teks). Namun perlu diperhatikan, browser WebKit yang lebih lama (sebelum Chrome 20 dan Safari 6) mengabaikan aturan ketika warna dihilangkan.
