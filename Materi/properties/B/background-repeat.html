background-repeat

(CSS : Properties)

Jika properti gambar latar ditentukan, properti pengulangan latar dalam CSS menentukan apakah (dan bagaimana) itu akan diulang. berikut adalah contoh penggunaannya :

# Syntax

CSS
html {

background-image : url (...) ;

background-repeat : repeat-x ;

}

Ini adalah nilai yang mungkin untuk properti ini (selain hal-hal yang biasa seperti warisan/inherit ) :

  • repeat:seperti lantai yang digambar di kedua arah. Ini adalah nilai default.
  • repeat-x: seperti lantai yang digambar secara horizontal.
  • repeat-y: seperti lantai yang digambar secara vertikal.
  • no-repeat: tidak diulang gambarnya, hanya sekali saja yang ditampilkan
  • space: seperti lantai yang digambar di kedua arah. Jangan pernah memotong gambar kecuali satu gambar terlalu besar untuk muat. Jika banyak gambar bisa pas, keluarkan dengan rata gambar yang selalu menyentuh tepi.
  • round: seperti lantai yang digambar di kedua arah. Jangan pernah memotong gambar kecuali satu gambar terlalu besar jika muat. Jika beberapa gambar bisa pas dengan ruang sisa, tekan atau regangkan untuk mengisi ruang. Jika kurang dari setengah lebar gambar tersisa, regangkan, jika lebih, regangkan.
