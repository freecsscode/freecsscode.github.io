top

(CSS : Properties)

Properti top memengaruhi posisi vertikal elemen yang diposisikan. Properti ini tidak berpengaruh pada elemen yang tidak diposisikan.

  • Jika posisi: absolut; atau posisi: tetap; - properti top menetapkan tepi atas elemen ke unit di atas / di bawah tepi atas leluhur yang diposisikan terdekat.
  • Jika posisi: relatif; - properti top membuat tepi atas elemen bergerak di atas / di bawah posisi normalnya.
  • Jika posisi: sticky; - properti top berperilaku seperti posisinya relatif ketika elemen berada di dalam viewport, dan seperti posisinya tetap ketika berada di luar.
  • Jika posisi: statis; - properti top tidak berpengaruh.

# Syntax

CSS

top : auto | length | % | initial | inherit ;



div {

top : 130px;

}

  • auto : Memungkinkan browser menghitung posisi tepi atas. Ini adalah standarnya.
  • length : Atur posisi tepi atas dalam px, cm, dll. Nilai negatif diperbolehkan.
  • % : Mengatur posisi tepi atas dalam% elemen yang mengandung.
  • initial : Setel properti ini ke nilai standarnya.
  • inherit : Mewarisi properti ini dari elemen induknya.
